# ResuForge

**ResuForge** is a **SaaS-based Resume Builder Application** that allows users to create, customize, and download professional resumes online with real-time editing, premium templates, and secure subscription management. The platform offers a seamless user experience with cloud storage, secure authentication, and direct resume email delivery to recruiters. Built using **React** and **Spring Boot**, ResuForge follows a scalable full-stack architecture with robust backend services.
